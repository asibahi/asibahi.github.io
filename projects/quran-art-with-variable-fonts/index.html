Quran Wall Art

Automatically justified Quran text using variable fonts.

This is two things:

  1. A practical application and exploration of the ideas discussed on Practical Arabic Justification
  2. Printed art pieces to give to family, friends, and clients.

There are multiple examples in the Github repo

I first posted about this on typo.social.

Born out of this project is my first (serious) Rust crate: kashida
